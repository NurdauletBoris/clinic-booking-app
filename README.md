# Clinic Booking App

Online platform for booking doctor appointments.

# Group Members
- Nurdaulet Boris
- Daniyar Saden


# Project Description
A full-stack web application built with Angular (frontend) 
and Django + DRF (backend) that allows patients to book 
appointments with doctors online.

# Tech Stack
- Frontend: Angular 17
- Backend: Django + Django REST Framework
- Auth: JWT (SimpleJWT)
- Database: PostgreSQL / SQLite
